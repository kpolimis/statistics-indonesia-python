Github tasks
forking a branch: fork a branch to your account 
add upstream remote: git remote add upstream [address.git]

commands we reviewed:
git add
git rm
git commit 
git status
git pull 
git push 
git checkout [branch]: switch to [branch]
git checkout -b [branch]: make a new branch called [branch]

steps to add data from local computer to repository
1. git status
   see the files you modified
2. git add/rm 
   add/delete the files you modified to staging 
3. git commit -m “[message]”
   commit the files you added/deleted for staging with message
4. git push 
   push files from local [branch] to repository [branch]
